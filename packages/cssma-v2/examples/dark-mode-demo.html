CSSMA v2 - Dark Mode Demo

Dark Mode with CSSMA v2

This demo shows how to use the dark mode utilities in CSSMA v2. You can switch between light mode, dark mode, system mode, or toggle the current mode.

The dark mode is implemented using the .dark class on the HTML element, which is the default strategy in CSSMA v2.

Custom Class Example

You can also use a custom class name for dark mode, such as .night-mode instead of .dark.
